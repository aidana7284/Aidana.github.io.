Covid Stats Project

This was a group project where we had to take information from an API in NYC Open Data and use objects as well as a for loop to tell the user a peice of information. My project takes information from the Covid statistics API in order to let the user put in a date then it will return the amount of Covid cases and Covid deaths on that specific date in NYC.

The main challenge I faced was making the if statement work. I had a hard time figuring how to make the user inputs work with the API so it would be able to call the right data set. I solved this problem by realizing the date needed to be in "2020-02-29T00:00:00.000" format so I prompted the user to put it into that format then added the "T00:00:00.000" to end during the if statement so that it would match one of the columns in the API.

My next steps are to continue working on my year long project using Babylon.js and also to continue practice with using APIs in javascript.
